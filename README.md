### 📚 Quick Start Guide

#### Initial Configuration

- **Login**: Connect to your **Stake** account directly via the application.
- **Settings**: Access settings to customize your experience according to your preferences.

#### Using the Code Claimer

1. Click on **"Claimer"** in the main interface.
2. Log in to **Stake** (required only on first use).
3. Leave the window open to allow automatic bonus code claiming.

#### Telegram Notifications

1. Go to the Telegram bot: [t.me/sspilot_bot](https://t.me/sspilot_bot)
2. Send the `/start` command to initiate the conversation.
3. Enter the code displayed in the application's spins interface to link your account.
